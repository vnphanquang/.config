# [polybar]

## Dependencies

| Dependency | Description |
| --- | --- |
| [rofi] | window launcher |
| [polybar gmail][polybar.gmail] | unread gmail notification (some python packages to install, see repo readme) |

> [!IMPORTANT]
> See [Rofi README](../rofi/README.md)

## Base Config

```bash
ln -sdf $HOME/dev/.config/polybar $HOME/.config/polybar
```
